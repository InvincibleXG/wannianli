# 📅 万年历 - H5 静态页面版

一个功能完整的本地万年历应用，支持 **1000年 - 2100年** 的精确节气历、农历和太阳历。

## ✨ 功能特性

### 核心功能
- ✅ **公历/农历互转** - 精确的日期转换
- ✅ **24节气** - 完整的节气表和节气日期
- ✅ **干支纪年** - 年月日时的干支信息
- ✅ **生肖** - 十二生肖查询
- ✅ **八字** - 八字和十神信息
- ✅ **宜忌** - 每日宜忌和吉凶信息
- ✅ **吉神方位** - 喜神、福神、财神方位
- ✅ **星宿** - 二十八星宿和彭祖百忌
- ✅ **节日** - 公历和农历节日
- ✅ **月历视图** - 完整的月份日历

### 技术特点
- 📦 **零依赖** - 完全离线使用，无需网络
- 🎨 **现代UI** - 深色主题，响应式设计
- ⚡ **高性能** - 纯前端实现，快速加载
- 📱 **移动友好** - 完美适配各种屏幕尺寸
- 🔍 **易用** - 直观的日期选择和快速导航

## 🚀 快速开始

### 方式一：直接打开（推荐）

```bash
# 进入项目目录
cd 万年历

# 用浏览器打开
open index.html

# 或者用 Python 启动本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 方式二：使用 Node.js 服务器

```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
http-server

# 访问 http://localhost:8080
```

### 方式三：使用 Live Server（VS Code）

1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎯 使用指南

### 基础操作

1. **选择日期**
   - 在顶部选择年、月、日
   - 选择历法类型（公历/农历）
   - 点击"查询"按钮

2. **快速导航**
   - **◀ 上年** - 跳转到上一年
   - **◀ 上月** - 跳转到上一个月
   - **今天** - 返回今天
   - **下月 ▶** - 跳转到下一个月
   - **下年 ▶** - 跳转到下一年

3. **月历视图**
   - 点击月历中的任意日期快速查询
   - 节气日期会高亮显示
   - 今天用特殊颜色标记

## 🔍 常见问题

### Q1: 为什么某些日期查不到？
**A**: 请检查日期是否有效。例如2月30日不存在。如果是1582年10月5-14日，这是历史上格里历改革删除的日期。

### Q2: 为什么农历和公历的对应关系有时不准确？
**A**: 农历计算基于天文学算法，某些边界日期可能存在1-2天的偏差。这是正常的。

### Q3: 可以离线使用吗？
**A**: 完全可以！这是一个纯前端应用，无需网络连接。

### Q4: 如何修改样式？
**A**: 编辑 `style.css` 文件。所有颜色、字体、布局都可以自定义。

### Q5: 如何添加更多功能？
**A**: 编辑 `app.js` 文件。lunar-javascript 库提供了丰富的 API，可以查询更多信息。
